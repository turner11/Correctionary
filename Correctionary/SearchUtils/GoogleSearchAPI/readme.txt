Google Search API for .NET 0.4 alpha
http://code.google.com/p/google-api-for-dotnet/


Description:

Provides a simple, unofficial, .NET Framework API for using Google Ajax Search API service.
This is a part of Google APIs for .NET project.

Feature:
- Support the Google Web Search API.
- Support the Google Local Search API.
- Support the Google News Search API.
- Support the Google Blog Search API.
- Support the Google Video Search API.
- Support the Google Book Search API.
- Support the Google Patent Search API.
- Support the Google Image Search API.
- CLS compatible. It can be used by any .NET languages (VB.NET, C++/CLI etc.)

Versions:

Google Search API for .NET comes in different versions for the various .NET frameworks.

-DotNet:
  .NET latest (3.5 SP1)

-DotNet20:
  .NET 2.0

-Silverlight:
  Silverlight 3.0

-Compact:
  Compact Framework 3.5
