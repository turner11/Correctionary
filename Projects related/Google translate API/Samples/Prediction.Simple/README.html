Instructions for the Google .NET Client API – Prediction.Simple

Browse Online

  • Browse Source, or main file Program.cs

1. Checkout Instructions

Prerequisites: Install Visual Studio or MonoDevelop, and Mercurial.

cd [someDirectory] 
hg clone https://code.google.com/p/google-api-dotnet-client.samples/ google-api-dotnet-client-samples

2. API access Instructions

Important: after checking out the project, and before compiling and running it, you need to:

  • Create a project in Google APIs console
  • Activate the Prediction API and Google Storage for your project
  • Edit ClientCredentials.cs to insert your client credentials which are generated on the Google APIs Console API Access pane.

3. Uploading a file containing prediction data

In order to predict something, you first have to upload a file containing prediction data and train the system. This project contains a simple sample set of data, language_id.txt, which you can upload to Google Storage.

Setup Project in Visual Studio

  • Open the GoogleApisSamples.sln with Visual Studio
  • Click on Build > Rebuild Solution
  • Execute the .exe in Prediction.Simple\bin\Debug

Setup Project in MonoDevelop

  • Open the GoogleApisSamples.sln with MonoDevelop
  • Click on Build > Rebuild All
  • Execute the .exe in Prediction.Simple\bin\Debug using mono
