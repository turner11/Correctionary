Instructions for the Google .NET Client API – GoogleApis.SampleHelper

This sample helper contains some useful methods which are shared between all samples. The code contained within here is usually not relevant for the samples themselves, but is used for nicer input/output and user interaction in general. The helper contains helper classes for:

  • Colored Console I/O
  • Requesting User Input
  • Parsing Command-Line Arguments
  • OAuth2 Token Caching (not recommended)
  • Reflection
  • General Utility and Extension methods
